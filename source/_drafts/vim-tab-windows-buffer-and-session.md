tabs:
:tabnew - Open a new tab

:tabedit <filename> - Edit the file with the provided name in a new tab

gt - Go to next tab open

gT - Go to previous tab

<Ctrl-w>T - Break the current window out to a new tab


windows:
Ctrl-w + s
Ctrl-w + v
